# HelloWorld

```C++
// C++ HelloWorld 模板
#include <iostream>

int main(int argc, char *argv[])
{
	std::cout << 0x0000 << " -> HelloWorld" << std::endl;
	return 0;
}
```

```Java
// Java HelloWorld 模板
public class HelloWorld {
	public static void main(String[] args) {
        System.out.println(0x0000 + " -> HelloWorld");
    }
}
```

```Python
// Python HelloWorld 模板
print(0x0000, "-> HelloWorld")
```
